# HackerNews

Anything that gratifies one's intellectual curiosity.

## Description
POC app for the hiring

## Installation

Use the package manager npm.

```bash
npm install
```
## Build
```bash
npm run build
```
Builds the application and creates bundle files

## Start Server 
```bash
npm start
```
Starts the server at```bash process.env.PORT || 3000```

## Test
```bash
npm run test
```
Runs the test files
